Welcome to Commandbar V1!

Some information:

-----ALL COMMANDS-----

help - lists all commands

file /create - creates a file

file /read - reads a file

file /write - writes text in a file

file /clear - clears a file

popup create /error - creates an error popup

popup create /warning - creates a warning popup

popup create /information - creates a information popup

popup create /question - creates a question popup

end - ends the session

viewinfo - shows version info

settings /templatechange - changes your python template
